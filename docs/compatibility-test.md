## 6. 발견된 이슈 및 해결

| 이슈 | 상태 | 해결 방법 |
|---|:---:|---|
| 모바일에서 드롭존 세로 배치 미적용 | ✅ 해결 | `grid` → `flex flex-col md:flex-row` 교체 |
| Safari에서 `document.cookie` 설정 후 즉시 미반영 | ✅ 해결 | `router.push()` 전 미들웨어 재검사로 처리 |
| iPad 채팅창 화면 밖으로 넘침 | ✅ 해결 | `w-80 sm:w-96` 고정폭 + `fixed` 포지셔닝으로 해결 |
